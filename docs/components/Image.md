**Image图片文档**

 1. props基本属性

| style               | 图片样式               |
| ------------------- | ---------------------- |
| source              | 图片来源               |
| circle              | boolean,是否为圆形图片 |
| maxWidth            | 图片最大宽度           |
| placeholderDisabled | 是否不需要占位图       |
|                     |                        |
|                     |                        |
|                     |                        |
|                     |                        |
|                     |                        |

 2. 基本用法
 

``` stylus
<Image source={{uri: this.props.poster}} placeholderDisabled style={s.poster} />
```
